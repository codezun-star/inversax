## Arquitectura del listado

El índice `/calculadoras` usa tags de categoría con filtrado client-side:
- **Tag activa por defecto:** primera categoría (Inversión)
- **Máximo visible por categoría:** 12. Si supera 12, aparece botón "Ver más" → `/calculadoras/[slug]`
- **Páginas de categoría:** `src/pages/calculadoras/[categoria].astro` (muestran TODAS, sin tope)
- **Contador dinámico del hero:** `grupos.reduce((s,g)=>s+g.calculadoras.length,0)`
- **Sitemap + breadcrumb JSON-LD:** autogenerados desde `grupos`

Para agregar una calculadora: (1) crear `src/pages/calculadoras/[slug].astro` y (2) añadir la entrada en `src/lib/calculadoras.ts`. Todo lo demás se actualiza solo.

## Patrón de formateo de moneda

```typescript
const DEC: Record<string,number> = {USD:2,COP:0,MXN:2,ARS:0,CLP:0,PEN:2,UYU:2,BOB:2,PYG:0,VES:2,CRC:0,DOP:2,GTQ:2,HNL:2,NIO:2,CUP:2,BZD:2,CAD:2,EUR:2};
function fmt(n: number, m: string) {
  const d = DEC[m] ?? 2;
  return new Intl.NumberFormat('es',{minimumFractionDigits:d,maximumFractionDigits:d}).format(n) + ' ' + m;
}
```
